# Python Data Structures and Strings

This repository contains two Python programs demonstrating dictionary operations and list slicing.

## Task 1: Create a Dictionary of Student Marks

- Creates a dictionary where student names are keys and their marks are values.
- Asks the user to input a student's name.
- Retrieves and displays the student's marks.
- Displays an error if the student is not found.

### Usage:

```sh
python student_marks.py
```
